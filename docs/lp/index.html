Designer Guide

Yumekano キャラクター編集ガイド

コードを書かずにキャラクターの性格や話し方を調整できます

目次

  1. キャラクター一覧を開く
  2. キャラクター詳細を確認する
  3. 編集ページを開く
  4. 各項目を編集する
  5. サンドボックスでテストする

1. キャラクター一覧を開く

ブラウザで http://localhost:3000/characters を開きます。

登録されているキャラクターがカード形式で表示されます

編集したいキャラクターをクリックしてください。

2. キャラクター詳細を確認する

キャラクターをクリックすると、詳細画面が開きます。

ペルソナ、スタイル、自律性、感情などが一覧で確認できます

表示される情報

セクション 内容
ペルソナ サマリー、価値観、欠点、好き/嫌いなもの
スタイル 敬語レベル、話し方のパラメータ、口癖
自律性 ユーザーの要求に対する反応傾向
感情ベースライン 基本的な感情状態 (PAD値)

3. 編集ページを開く

ステップ1: ワークスペースタブをクリック

ワークスペース一覧が表示されます

ステップ2: 「編集」ボタンをクリック

ワークスペース（Default Workspace）の「編集」ボタンをクリックします。

4. 各項目を編集する

編集ページでは、タブを切り替えて各項目を編集できます。

アイデンティティ

キャラクターの基本情報を設定します。

表示名、年齢、一人称、二人称、職業を設定
項目説明
表示名キャラクターの名前
年齢設定上の年齢
一人称自分のことをどう呼ぶか（わたし、僕、など）
二人称ユーザーをどう呼ぶか（○○さん、きみ、など）
職業・役割何をしている人か

ペルソナ

キャラクターの性格を設定します。

価値観、欠点、好きなもの等をタグ形式で編集
タグの追加: テキストを入力して「追加」ボタンをクリック（またはEnterキー）
タグの削除: タグの「×」をクリック

スタイル

話し方のスタイルを設定します。

スライダーで細かく調整できます
項目低い場合高い場合
簡潔さ丁寧に説明短く端的に
直接性遠回しにストレートに
遊び心真面目に冗談を交えて
からかいからかわないよくからかう
主導性受け身積極的に話題を振る

プロンプト（上級者向け）

AIへの指示文を直接編集できます。

Generator, Planner, Extractor, Reflector, Ranker
注意: プロンプトの編集は上級者向けです。わからない場合は触らなくてOKです。

5. サンドボックスでテストする

編集した内容をすぐにテストできます。編集ページ右上の「サンドボックス」ボタンをクリック。

サンドボックス画面

メッセージを入力して「送信」をクリック。

キャラクターが応答します

表示される情報

項目説明
フェーズ現在の関係段階
PAD感情状態（P=快楽、A=覚醒、D=支配）
TurnターンID（デバッグ用）

編集の流れまとめ

キャラクター一覧 → 詳細画面 → ワークスペース → 編集ページ → サンドボックス

よくある質問

Q: 変更を保存するには？

各タブで編集後、画面下の「保存」ボタンをクリックしてください。

Q: 元に戻したい場合は？

ページを更新（リロード）すると、最後に保存した状態に戻ります。

Q: テストした会話をリセットしたい場合は？

サンドボックス画面右上の「リセット」ボタンをクリックしてください。
