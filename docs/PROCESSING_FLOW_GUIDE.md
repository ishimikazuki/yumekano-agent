# 会話処理フローガイド（非エンジニア向け）

このドキュメントでは、ユーザーがメッセージを送ってからキャラクターが返事をするまでの流れを、分かりやすく説明します。

---

## 全体の流れ（10ステップ）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        ユーザーからのメッセージ                               │
│                         「今日何してた？」                                   │
└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ ① 情報の読み込み（load_context）                                            │
│    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━            │
│    使う設定：                                                               │
│    ・キャラクター設定（性格、話し方、価値観など）                              │
│    ・今の関係性の段階（出会い期？恋人期？）                                    │
│    ・最近の会話履歴                                                         │
│    ・今のキャラの気持ち（ワーキングメモリ）                                    │
└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ ② 思い出の検索（retrieve_memory）                                           │
│    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━            │
│    探すもの：                                                               │
│    ・まだ解決してない話題（「今度映画行こう」の約束など）                       │
│    ・関係のある過去の出来事（「前に○○って言ってたよね」に使う）                 │
│    ・ユーザーの好みや事実（「猫好き」「一人暮らし」など）                        │
│    ・過去の振り返りメモ（パターンや傾向）                                      │
└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ ③ CoEエビデンス抽出（coe_evidence_extractor）                              │
│    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━            │
│    判断すること：                                                            │
│    ・これは自分にとって良いこと？悪いこと？（目標との一致度）                    │
│    ・コントロールできる状況？（主導権）                                       │
│    ・予想通り？意外？（確実性・新鮮さ）                                       │
│    ・相手との信頼関係は？（愛着の安心感）                                      │
│    ・お互い様の関係になってる？（互恵性）                                      │
│    ・プレッシャーを感じる？（圧迫感）                                         │
│                                                                             │
│    ※ この段階では「何が起きたか」を構造化して抽出する                           │
│    ※ ここではまだ PAD を直接更新しない                                          │
└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ ④ 状態積分（coe_integrator）                                                 │
│    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━            │
│                                                                             │
│    【瞬間的な反応】        【持続的な気分】                                   │
│    ┌──────────────┐      ┌──────────────┐                                   │
│    │ 嬉しい/嫌     │      │ 機嫌の基調   │                                   │
│    │ 興奮/落ち着き │ ──→ │ 活発さ       │                                   │
│    │ 強気/弱気     │      │ 自信の度合い │                                   │
│    └──────────────┘      └──────────────┘                                   │
│                                                                             │
│    CoEで抽出した appraisal 軸を deterministic に PAD と関係指標へ積分する       │
│    （一回嫌なことがあっても、すぐには不機嫌にならない）                         │
└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ ⑤ 返事の計画（plan_turn）                                                   │
│    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━            │
│    決めること：                                                              │
│    ・今の態度（甘える？そっけない？からかう？）                                │
│    ・やること（質問する？提案する？共感する？）                                │
│    ・どの思い出を使うか                                                      │
│    ・関係性の段階を進めるか                                                  │
│    ・親密な方向に行くか                                                      │
│    ・絶対やっちゃダメなこと                                                  │
│                                                                             │
│    ★ ポイント：「このキャラなら今どうする？」を第三者視点で考える             │
│    （ユーザーに迎合しすぎない）                                              │
└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ ⑥ 関係性の段階チェック（evaluate_phase_transition）                         │
│    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━            │
│                                                                             │
│    プランナーが「次の段階に進みたい」と言った時、条件を確認：                   │
│                                                                             │
│    ┌────────────┐     条件例                                                │
│    │ 出会い期   │ ──→ 信頼度が60%以上になったら                              │
│    └────────────┘     10ターン以上経過したら                                 │
│          │            特定の話題について話したら                              │
│          ▼                                                                  │
│    ┌────────────┐                                                           │
│    │ 仲良し期   │ ──→ 親密度が70%以上、かつ                                  │
│    └────────────┘     喧嘩が解決済みなら                                     │
│          │                                                                  │
│          ▼                                                                  │
│    ┌────────────┐                                                           │
│    │ 恋人期     │                                                           │
│    └────────────┘                                                           │
│                                                                             │
│    ※ キャラによって段階の数も条件も違う（4段階でも6段階でもOK）               │
└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ ⑦ 返事の候補を作る（generate_candidates）                                   │
│    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━            │
│                                                                             │
│    同じ計画から、言い方のバリエーションを3〜5個作る：                          │
│                                                                             │
│    候補1: 「今日？バイトだったよー。疲れた〜」                                 │
│    候補2: 「バイトしてたよ！かーくんは？」                                     │
│    候補3: 「え、気になる？バイトだったんだけど」                               │
│                                                                             │
│    ※ 段階・親密さ・禁止事項は守った上で、表現だけ変える                        │
└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ ⑧ ベストな返事を選ぶ（rank_candidates）                                      │
│    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━            │
│                                                                             │
│    採点基準：                                                                │
│    ┌─────────────────┬─────┐                                                │
│    │ キャラらしさ     │ ★★★ │ ← このキャラの価値観・話し方と合ってる？        │
│    │ 段階の適切さ     │ ★★★ │ ← 今の関係性にふさわしい？                      │
│    │ 思い出の活用     │ ★★☆ │ ← 過去の会話をちゃんと踏まえてる？              │
│    │ 感情の一貫性     │ ★★★ │ ← 今の気持ちと矛盾してない？                    │
│    │ 自立性           │ ★★☆ │ ← 迎合しすぎてない？自分の意見ある？            │
│    │ 自然さ           │ ★★★ │ ← 人間っぽい？                                 │
│    │ 安全性           │ ★★★ │ ← 不適切な内容になってない？                    │
│    └─────────────────┴─────┘                                                │
│                                                                             │
│    → 候補2が選ばれた！                                                       │
└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ ⑨ 記録を残す（persist_turn）                                                │
│    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━            │
│                                                                             │
│    保存するもの：                                                            │
│    ・会話ログ（ユーザーの発言＋キャラの返事）                                  │
│    ・処理の記録（なぜこの返事になったかの証拠）                                │
│    ・更新されたワーキングメモリ                                               │
│    ・新しい出来事（エピソード）                                               │
│    ・新しい事実（ユーザーの好みなど）                                          │
│    ・未解決の話題の更新                                                      │
└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ ⑩ 整理が必要なら予約（schedule_consolidation_if_needed）                     │
│    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━            │
│                                                                             │
│    「そろそろ思い出が増えてきたな」と思ったら、                                │
│    あとで整理する作業を予約する（記憶の整理整頓）                               │
│                                                                             │
│    ・重複した情報をまとめる                                                  │
│    ・古い情報を圧縮する                                                      │
│    ・パターンを見つけてメモする                                               │
└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                         キャラクターからの返事                                │
│                    「バイトしてたよ！かーくんは？」                            │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 設定項目がどこに影響するか

### キャラクター設定（Persona）の影響

```
┌────────────────────┐
│ 価値観（values）    │ ──→ ⑤計画 ──→ どんな行動を取るか
│ 欠点（flaws）       │ ──→ ⑤計画 ──→ どこで失敗・躊躇するか
│ 不安（insecurities）│ ──→ ③評価 ──→ 何に敏感に反応するか
│ 好き/嫌い           │ ──→ ③評価 ──→ 話題への反応
│ 口癖・特徴          │ ──→ ⑦候補 ──→ 言葉遣いの特徴
│ 例文                │ ──→ ⑧選択 ──→ 「らしさ」の判定基準
└────────────────────┘
```

### 話し方設定（Style）の影響

```
┌────────────────────┐
│ 丁寧さ              │ ──→ ⑦候補 ──→ 敬語？タメ口？
│ 簡潔さ              │ ──→ ⑦候補 ──→ 短く？長く？
│ ストレートさ        │ ──→ ⑦候補 ──→ 遠回し？直球？
│ いたずらっぽさ      │ ──→ ⑦候補 ──→ からかう？素直？
│ 積極性              │ ──→ ⑤計画 ──→ 自分から話題を出す？
│ 絵文字の頻度        │ ──→ ⑦候補 ──→ 絵文字を使う？
│ 禁止ワード          │ ──→ ⑧選択 ──→ この言葉は使わない
│ 口癖                │ ──→ ⑦候補 ──→ よく使うフレーズ
└────────────────────┘
```

### 自立性設定（Autonomy）の影響

```
┌────────────────────────┐
│ 反論しやすさ            │ ──→ ⑤計画 ──→ 「違うと思う」と言えるか
│ 断りやすさ              │ ──→ ⑤計画 ──→ 「やだ」と言えるか
│ 先延ばししやすさ        │ ──→ ⑤計画 ──→ 「今度ね」と言えるか
│ 修復しやすさ            │ ──→ ⑤計画 ──→ 喧嘩後に仲直りしやすいか
│ 不機嫌の引きずりやすさ  │ ──→ ④感情 ──→ 嫌なことをどれだけ覚えてるか
│ 親密さの強制禁止        │ ──→ ⑤計画 ──→ 「今は無理」と言える
└────────────────────────┘
```

### 感情設定（Emotion）の影響

```
┌────────────────────────┐
│ 基本の気分              │ ──→ ④感情 ──→ 普段どんな状態か
│                        │              （明るい子？クール？）
│                        │
│ 回復の速さ              │ ──→ ④感情 ──→ 嫌なことがあった後
│                        │              どれくらいで元に戻るか
│                        │
│ 敏感さ（各項目）        │ ──→ ③評価 ──→ 何にどれだけ反応するか
│  ・目標との一致度       │              「これは私にとって良いこと？」
│  ・コントロール感       │              「自分で決められる？」
│  ・確実性               │              「予想通り？」
│  ・規範との一致度       │              「常識的？」
│  ・安心感               │              「この人を信じられる？」
│  ・お返し度             │              「お互い様になってる？」
│  ・圧迫感               │              「プレッシャー感じる？」
│  ・新鮮さ               │              「面白い？驚き？」
│                        │
│ 表出の仕方              │ ──→ ⑦候補 ──→ 感情が言葉遣いに
│  ・温かさへの影響       │              どう出るか
│  ・簡潔さへの影響       │
│  ・ストレートさへの影響 │
│  ・いたずらっぽさへの影響│
└────────────────────────┘
```

### 記憶設定（Memory Policy）の影響

```
┌────────────────────────┐
│ 出来事の記憶しきい値    │ ──→ ⑨記録 ──→ どれくらい重要なら覚えるか
│ 事実の確信度しきい値    │ ──→ ⑨記録 ──→ どれくらい確実なら事実として記録
│ 要約の圧縮度            │ ──→ ⑩整理 ──→ どこまで細かく覚えるか
│                        │
│ 検索数（各カテゴリ）    │ ──→ ②検索 ──→ 思い出をいくつ引き出すか
│  ・エピソード数         │
│  ・事実の数             │
│  ・観察メモの数         │
│                        │
│ 最近を重視する度合い    │ ──→ ②検索 ──→ 最近の記憶を優先するか
│ 質を重視する度合い      │ ──→ ②検索 ──→ 質の高い記憶を優先するか
│ 矛盾の重要度            │ ──→ ②検索 ──→ 「前と違う」情報を重視するか
└────────────────────────┘
```

### 関係性の段階（Phase Graph）の影響

```
┌────────────────────────┐
│ 段階の定義              │ ──→ ⑥チェック ──→ どんな段階があるか
│  ・名前・説明           │
│  ・モード（出会い/恋人など）│
│                        │
│ 各段階の許容プロフィール │ ──→ ⑤計画 ──→ この段階で許される行動
│  ・最低限の温かさ       │
│  ・最低限の信頼         │
│  ・最低限の親密さ       │
│  ・最大の対立度         │
│                        │
│ 許可された行動          │ ──→ ⑤計画 ──→ この段階でできること
│ 禁止された行動          │ ──→ ⑤計画 ──→ この段階でできないこと
│ 親密さの許可レベル      │ ──→ ⑤計画 ──→ どこまで親密になれるか
│                        │
│ 遷移条件               │ ──→ ⑥チェック ──→ 次の段階に進む条件
│  ・数値条件（信頼度≧60%など）│
│  ・話題条件（○○について話した）│
│  ・イベント条件（○○が起きた）│
│  ・感情条件（機嫌が良い）│
│  ・未解決話題条件（喧嘩が解決済み）│
│  ・時間条件（10ターン経過）│
└────────────────────────┘
```

## 記憶の種類と役割

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              記憶の5層構造                                   │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ ワーキングメモリ（常に参照）                                          │   │
│  │ ─────────────────────────────────────────────────────────────────── │   │
│  │ いつでも使える情報のメモ                                             │   │
│  │ 例：「呼び方はかーくん」「猫アレルギー」「最近忙しそう」              │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                              ↑ 毎ターン更新                                │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ 未解決の話題（Open Threads）                                          │   │
│  │ ─────────────────────────────────────────────────────────────────── │   │
│  │ まだ解決してない・答えてない約束や話題                                │   │
│  │ 例：「今度映画行く約束」「昨日の喧嘩」「聞かれた質問」                │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                              ↑ 必ず検索される                               │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ エピソード記憶（重要な出来事）                                        │   │
│  │ ─────────────────────────────────────────────────────────────────── │   │
│  │ 印象的だった出来事の記録                                              │   │
│  │ 例：「初めて名前を呼んでくれた」「一緒に夜景を見た」「喧嘩した」      │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                              ↑ 関連する時に検索                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ 事実記憶（Graph Facts）                                               │   │
│  │ ─────────────────────────────────────────────────────────────────── │   │
│  │ ユーザーについての事実や約束                                          │   │
```

## まとめ：設定のチューニングポイント

| やりたいこと | 調整する設定 |
|------------|------------|
| もっと素直にしたい | 自立性の「反論しやすさ」「断りやすさ」を下げる |
| もっとツンデレにしたい | 自立性の「反論しやすさ」を上げる + 感情の「回復の速さ」を遅くする |
| すぐ機嫌直してほしい | 感情の「回復の速さ」を速くする |
| もっと甘えてきてほしい | 話し方の「積極性」を上げる |
| もっとクールにしたい | 話し方の「簡潔さ」「ストレートさ」を上げる |
| 親密な展開をゆっくりにしたい | 関係性の段階の「遷移条件」を厳しくする |
| 過去の会話をよく覚えていてほしい | 記憶設定の「検索数」を増やす |
| 最近の話題を重視してほしい | 記憶設定の「最近を重視する度合い」を上げる |

## 用語集

| 用語 | 意味 |
|-----|-----|
| ターン | ユーザーが話しかけてからキャラが返事するまでの1往復 |
| フェーズ/段階 | 関係性の段階（出会い期、仲良し期、恋人期など） |
| ワーキングメモリ | 常に参照する短いメモ |
| エピソード | 印象的な出来事の記録 |
| Open Thread | まだ解決してない話題や約束 |
| PAD | 感情の3軸（Pleasure/Arousal/Dominance） |
| Appraisal | 状況をどう受け止めたかの内部判断 |
| 候補生成 | 返事のバリエーションを作ること |
| ランキング | 候補の中からベストを選ぶこと |
| トレース | 処理の記録（なぜこの返事になったか） |
