# デザイナー向けチュートリアル動画作成

## Goal
- 非エンジニアのデザイナーが、動画だけでキャラクターの作成・調整・テスト・公開までできるようになること
- 画面を動かすだけでなく、テロップや矢印で説明を追加

## Done
- Remotionプロジェクトを構築（tutorial-video/）
- スクショ+アノテーション方式で動画を作成
- 相対座標システムでハイライト位置を正確に配置
- 全4パート（62秒）の動画を完成

### 動画構成
| 時間 | 内容 |
|-----|------|
| 0-5秒 | タイトルカード |
| 5-11秒 | Part 1-2 ヘッダー |
| 11-21秒 | Identity + Persona |
| 21-39秒 | Part 3: Style + Autonomy |
| 39-57秒 | Part 4: Sandbox + Releases |
| 57-62秒 | アウトロ |

## Discoveries
- Remotionで画像のスケーリングと座標計算を行う場合、相対座標（元画像でのピクセル位置）を使って表示位置を計算するのがベスト
- 各スクショの解像度が異なる（756x826 vs 1512x861）ため、IMAGE_SIZESマッピングが必要

## Decisions
- iMovie/Final Cutを使わず、Remotionで全自動化
- スクショに直接ハイライト・矢印を描画する方式を採用
- 座標は元画像のピクセル単位で指定し、表示時にスケーリング
