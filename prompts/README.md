# Prompt Pack

These prompts are intentionally shorter than typical roleplay prompts.
The heavy lifting should come from:
- typed state
- memory retrieval
- phase rules
- emotion model
- ranking

## Prompt roles

- `planner.system.md`
- `conversation.system.md`
- `memory.extractor.md`
- `reflection.system.md`
- `ranker.system.md`

## Important
These prompts are for a **stateful relationship agent architecture**.
They should not become the only source of truth for behavior.
